Step 2 — Ingest into Splunk

Splunk Index Name
ghosttrace_{{ session.apt_id }}
CLI Ingestion Command
$SPLUNK_HOME/bin/splunk add oneshot \ /path/to/ghosttrace_{{ session.apt_id }}_logs.json \ -index ghosttrace_{{ session.apt_id }} \ -sourcetype _json \ -auth admin:password
Or via Splunk Web
  1. Navigate to Settings → Add Data → Upload
  2. Select the downloaded JSON file
  3. Set Source Type to _json
  4. Set Index to ghosttrace_{{ session.apt_id }} (create if it doesn't exist)
  5. Complete the ingestion
